# DocuVault

DocuVault is a secure document vault. It protects files with per-document
AES encryption, enforces policy-driven access control, and keeps a tamper-evident
audit trail of every action taken on a protected document.

## Features

- **Protect on upload** — every document is encrypted with its own key the moment
  it enters the vault; keys live in a dedicated key store, never beside the data.
- **Policy-based sharing** — owners share documents with other users; every open
  is checked against the active access policy at that moment.
- **Instant revocation** — revoking access rotates the document key, so previously
  shared copies can no longer be opened.
- **Compliance reporting** — every upload, open, share, revoke, and delete is
  recorded and can be exported as a compliance report.

## Architecture

```
api/           entry points (controllers) — one handler per user action
service/       business logic: documents, access, policy, users, audit, notifications
crypto/        encryption engine, key manager, cipher utilities
repository/    persistence for documents, users, policies, audit events, keys
model/         domain objects
util/          validation, file helpers, logging
```

A typical request travels: `api` → `service` → (`crypto` | `repository`) with the
audit service recording the outcome.
